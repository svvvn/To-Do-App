**SVN Todo**

  A modern and interactive Todo application built using pure HTML, CSS, and Vanilla JavaScript.
  Designed with a clean glassmorphism UI and smooth animations to make daily task management simple and enjoyable.


**Overview**
  
  SVN Todo is a lightweight task manager that allows users to:
  
  Add new tasks
  
  Mark tasks as completed
  
  Delete tasks
  
  Track completion progress visually
  
  Persist tasks using LocalStorage

The app focuses on clean UI design and solid JavaScript fundamentals without relying on any frameworks.


**Features**

  Real-time progress bar calculation
  
  Live statistics (Total / Completed tasks)
  
  Persistent storage using LocalStorage
  
  Smooth add/remove animations
  
  Animated motivational background
  
  Fully responsive layout
  
  RTL support


**Tech Stack**

  HTML5
  
  CSS3 (Glassmorphism, animations, gradients)
  
  Vanilla JavaScript
  
  Font Awesome Icons
  
  LocalStorage API


**Implementation Highlights**

State management using a centralized tasks array

Re-render pattern to update UI consistently

Dynamic DOM manipulation

Event-driven interactions

Percentage-based progress calculations
